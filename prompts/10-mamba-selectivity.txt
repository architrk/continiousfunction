4. The "Selective Copying" task: input has a password surrounded by noise, output should copy only the password
5. Compare LTI (Linear Time Invariant) vs Selective: LTI fails, Mamba succeeds
6. Animate the parallel scan algorithm (optional advanced view)

Interactive: toggle different input patterns to see how selectivity adapts.
