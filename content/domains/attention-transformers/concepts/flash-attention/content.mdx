## Intuition

Standard attention is conceptually simple but computationally awkward on GPUs:

1. Compute the $T\times T$ score matrix $S=QK^\top/\sqrt{d_k}$.
2. Apply a row-wise softmax to get attention weights $P=\mathrm{softmax}(S)$.
3. Multiply by values: $PV$.

The problem is step (1): materializing a $T\times T$ matrix is huge, and moving it to and from GPU memory (HBM) is slow. For long sequences, attention is often **memory-bandwidth bound** rather than FLOPs bound.

FlashAttention is the idea: keep computations in fast on-chip SRAM by tiling, and compute the softmax in a streaming way so you never need to store the full $T\times T$ matrix.

## Math

### Standard attention (and the $T\times T$ issue)

For one head, with $Q,K,V\in\mathbb R^{T\times d_k}$:

$$
\mathrm{Attn}(Q,K,V)=\mathrm{softmax}\!\left(\frac{QK^\top}{\sqrt{d_k}}\right)V.
$$

The attention weights matrix is $P\in\mathbb R^{T\times T}$. If you store it explicitly, memory scales like $O(T^2)$.

### Online softmax (streaming trick)

For a single row of scores $s\in\mathbb R^T$, the softmax uses:

$$
\mathrm{softmax}(s)_j = \frac{\exp(s_j-m)}{\sum_{k=1}^T \exp(s_k-m)},\qquad m=\max_k s_k.
$$

You can compute this in blocks without ever storing all scores at once by maintaining a running max $m$ and a running normalizer $\ell$.

For a block $b$ of scores with:

$$
m_b=\max(s_b),\qquad \ell_b=\sum_{j\in b}\exp(s_j-m_b),\qquad o_b=\sum_{j\in b}\exp(s_j-m_b)\,v_j,
$$

you can merge block statistics with the running state via:

$$
m\leftarrow \max(m,m_b),
$$
$$
\ell\leftarrow e^{m_{old}-m}\,\ell_{old}+e^{m_b-m}\,\ell_b,
$$
$$
o\leftarrow e^{m_{old}-m}\,o_{old}+e^{m_b-m}\,o_b.
$$

At the end, the attention output for that row is:

$$
\mathrm{out}=\frac{o}{\ell}.
$$

This is the core idea: you can compute $PV$ without ever materializing $P$.

## Code

```python
import numpy as np

def softmax(x):
    m = float(x.max())
    e = np.exp(x - m)
    return e / e.sum()

def attn_out_stream(scores, values, block=2):
    m, l, o = -np.inf, 0.0, 0.0
    for i in range(0, len(scores), block):
        sb = scores[i:i+block]
        vb = values[i:i+block]
        mb = float(sb.max())
        eb = np.exp(sb - mb)
        lb = float(eb.sum())
        ob = float((eb * vb).sum())
        m_new = max(m, mb)
        l = np.exp(m - m_new) * l + np.exp(mb - m_new) * lb
        o = np.exp(m - m_new) * o + np.exp(mb - m_new) * ob
        m = m_new
    return o / l

s = np.array([2.0, 1.0, -1.0, 0.5, 3.2, -0.3])
v = np.array([0.1, 0.2, -0.4, 0.0, 0.7, 0.3])

out_full = float((softmax(s) * v).sum())
out_stream = float(attn_out_stream(s, v, block=2))
print("full:", round(out_full, 6), "stream:", round(out_stream, 6), "diff:", round(abs(out_full - out_stream), 9))
```
