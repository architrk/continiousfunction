## Intuition

Self-attention is a learnable way to *copy information* across a sequence.

Each token produces:

- a **query**: what am I looking for?
- a **key**: what do I contain?
- a **value**: what content should be copied if you pick me?

Attention compares a query to all keys, turns those similarities into a probability distribution, then takes a weighted average of values. The output is a new representation for the token: "what I know after looking at everyone else."

Multi-head attention runs several such lookups in parallel so the model can learn multiple kinds of "what to look for" at once (syntax, long-range dependencies, coreference, etc.).

## Math

Given a sequence represented as a matrix $X \\in \\mathbb{R}^{T\\times d_{\\text{model}}}$, form:

$$Q = XW_Q,\\quad K = XW_K,\\quad V = XW_V.$$

Single-head attention is:

$$\\mathrm{Attn}(Q,K,V) = \\mathrm{softmax}\\!\\left(\\frac{QK^\\top}{\\sqrt{d_k}}\\right)V.$$

For causal (autoregressive) language modeling, apply a mask $M$ where $M_{ij}=-\\infty$ for $j>i$:

$$\\mathrm{Attn}(Q,K,V) = \\mathrm{softmax}\\!\\left(\\frac{QK^\\top + M}{\\sqrt{d_k}}\\right)V.$$

Multi-head attention computes several heads and mixes them:

$$\\mathrm{MHA}(X) = \\mathrm{Concat}(h_1,\\dots,h_H)W_O,\\quad h_i = \\mathrm{Attn}(XW_Q^{(i)}, XW_K^{(i)}, XW_V^{(i)}).$$

A standard transformer block (pre-norm) is:

$$\\begin{aligned}
H' &= H + \\mathrm{MHA}(\\mathrm{LN}(H)) \\\\
H^{\\text{out}} &= H' + \\mathrm{MLP}(\\mathrm{LN}(H'))
\\end{aligned}$$

The core cost driver is the $T\\times T$ attention matrix: compute and memory scale roughly as $O(T^2)$.

## Code

```python
import numpy as np

def softmax(x):
    x = x - x.max(axis=-1, keepdims=True)
    ex = np.exp(x)
    return ex / ex.sum(axis=-1, keepdims=True)

rs = np.random.RandomState(0)
T, d = 5, 4
X = rs.randn(T, d)
Wq, Wk, Wv = rs.randn(d, d), rs.randn(d, d), rs.randn(d, d)

Q, K, V = X @ Wq, X @ Wk, X @ Wv
scores = (Q @ K.T) / np.sqrt(d)

mask = np.triu(np.ones((T, T)), k=1) * (-1e9)  # causal mask
weights = softmax(scores + mask)
O = weights @ V

print("attention weights (rows sum to 1):")
print(np.round(weights, 3))
print("output:")
print(np.round(O, 3))
```
