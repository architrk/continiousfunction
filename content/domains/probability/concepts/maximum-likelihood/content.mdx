## Intuition

Maximum likelihood is the simplest way to fit a probabilistic model: choose parameters that make the data you saw as unsurprising as possible.

If your model says your dataset was extremely unlikely, it’s probably the wrong model (or the wrong parameters). MLE just turns that intuition into an objective you can optimize.

In deep learning, this becomes: train the model to assign high probability to the training data. For language models, that’s “predict the next token well,” which is cross-entropy.

## Math

Given i.i.d. data $x^{(1)},\dots,x^{(n)}$ and a parametric model $p_\theta(x)$, the **likelihood** is

$$L(\theta) = \prod_{i=1}^n p_\theta(x^{(i)}).$$

We maximize log-likelihood:

$$\ell(\theta) = \log L(\theta) = \sum_{i=1}^n \log p_\theta(x^{(i)}).$$

Equivalently, we minimize **negative log-likelihood** (NLL):

$$\min_\theta -\frac{1}{n}\sum_{i=1}^n \log p_\theta(x^{(i)}).$$

If $\hat p$ is the empirical data distribution, this is exactly **cross-entropy**:

$$H(\hat p, p_\theta) = -\mathbb{E}_{x\sim \hat p}\,[\log p_\theta(x)].$$

And since

$$H(\hat p, p_\theta) = H(\hat p) + \mathrm{KL}(\hat p\,\|\,p_\theta),$$

minimizing cross-entropy is the same as minimizing $\mathrm{KL}(\hat p\,\|\,p_\theta)$ (because $H(\hat p)$ doesn’t depend on $\theta$).

## Code

```python
import numpy as np

# MLE for a 1D Gaussian mean (sigma known)
np.random.seed(0)
mu_true, sigma = 2.0, 1.5
x = mu_true + sigma * np.random.randn(2000)

mu_mle = x.mean()
print("mu_true=", mu_true, "mu_mle=", mu_mle)

# Bernoulli MLE: p_hat = fraction of 1s
y = (np.random.rand(5000) < 0.3).astype(float)
print("p_mle=", y.mean())
```
