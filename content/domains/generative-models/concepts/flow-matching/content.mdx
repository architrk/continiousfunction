## Intuition

Diffusion models teach a model to "undo noise" step by step. Flow matching teaches a model something more direct: **which direction to move**.

Imagine starting with a cloud of noise points and wanting to morph it into a cloud shaped like your data. If you knew a time-dependent "wind field" $v(x,t)$ that pushes particles, you could integrate an ODE and watch noise turn into samples.

The first idea to get right is the supervised label. For one chosen noise/data pair, a straight conditional path uses the current point $x_t=(1-t)x_0+t x_1$ but trains on the constant velocity $u_t=x_1-x_0$. The arrow from $x_t$ to $x_1$ is only the remaining displacement, not the training target.

## Math

### Generative flow and regression target

We model a trajectory $x_t$ with a neural velocity field $v_\theta$, then train that field to match a target velocity $u_t$:

$$
\frac{dx_t}{dt}=v_\theta(x_t,t),\qquad
\mathcal L=\mathbb E_{t,x_0,x_1}\,\big\|v_\theta(x_t,t)-u_t\big\|^2.
$$

### Rectified flow (straight paths)

Choose a straight interpolation between noise $x_0$ and data $x_1$. Differentiating the path gives the supervised velocity label:

$$
x_t=(1-t)x_0+t x_1,\qquad
u_t=\frac{d}{dt}x_t=x_1-x_0,\qquad
x_1-x_t=(1-t)(x_1-x_0).
$$

The remaining displacement $x_1-x_t$ is shorter than the training label except at $t=0$.

So the useful identity is:

$$
x_t + (1-t)u_t = x_1.
$$

For a single straight conditional pair, the velocity label stays fixed as $t$ changes; the remaining integration time shrinks.

## Code

```python
import numpy as np

rng = np.random.default_rng(0)
x0 = rng.standard_normal(5)

# A simple deterministic "data" mapping for the toy example
x1 = 2.0 * x0 + 1.0

for t in [0.0, 0.25, 0.5, 0.75, 1.0]:
    xt = (1 - t) * x0 + t * x1
    u = x1 - x0
    remaining = x1 - xt

    # For this toy mapping, the exact velocity field can be written as v(xt,t):
    # xt = (1+t)x0 + t  =>  x0 = (xt - t)/(1+t)  =>  u = x0 + 1 = (xt + 1)/(1+t)
    v = (xt + 1.0) / (1.0 + t)
    print(
        "t=", t,
        "max|u-v|=", float(np.max(np.abs(u - v))),
        "remaining/full=", round(float(np.linalg.norm(remaining) / np.linalg.norm(u)), 2),
    )
```
